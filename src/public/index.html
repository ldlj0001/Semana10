Concepto

Es un sistema de bases de datos relacionales, esto quiere decir uqe su estructura de datos depende de la relación existente entre sus diferentes tablas. . MySql tiene dos versiones, una que es de código abierto, y otra versión comercial que es gestionada por la compañía Oracle.

Comandos Básicos desde consola de comandos

Login

Mysql -h localhost -u User - pPassword

Create Database

CREATE DATABASE db_name;

Create Table

CREATE TABLE table_name ( column1 datatype, column2 datatype, column3 datatype, .... );

Insert Data

INSERT INTO table_name (column1, column2, column3, ...) VALUES (value1, value2, value3, ...);

Query Data

SELECT column1, column2, ... FROM table_name;

Concepto

MongoDb usa un esquema de bases de datos no relacionales, lo cual quiere decir que no asocia la informaciòn basada en tablas interconectadas, sino en esquema de datos BSON (Basado en JSON, de hecho la informaciòn en este tipo de base se inserta usando un JSON), que permite anidar en un mismo "Documento" la data necesaria para el registro insertado.

Comandos Básicos desde consola de comandos

Login

mongo -u AdminSammy -p --authenticationDatabase admin

Create Database

use [database_name]

Create Table

db.createCollection(name, options)

Insert Data

db.inventory.insertOne( { item: "canvas", qty: 100, tags: ["cotton"], size: { h: 28, w: 35.5, uom: "cm" } } )

Query Data

db.collection.find(query, projection, options)

Concepto

Es una base de datos relacional orientada a objetos, lo cual permite que se puedan crear nuevos tipos de datos que se ajusten a lo que se requiera. Dentro de sus caracteristicas principales se encuentran robustez de funciones, alta fiabilidad, rendimiento, flexibilidad y facilidad de replicación.

Comandos Básicos desde consola de comandos

Login

psql -U testuser testdb

Create Database

sudo -u postgres createdb --owner=testuser testdb

Create Table

testdb=> create table if not exists cafe ( name varchar(50), price numeric(5,2), primary key (name)); CREATE TABLE

Insert Data

testdb=> insert into cafe values ('Espresso', 3.99), ('Green Tea', 2.99);

Query Data

SELECT select_list FROM table_name;
